# 🎙️ Voice AI Bank Assistant (STT -> LLM -> TTS)

Интерактивный голосовой ИИ-ассистент службы поддержки банка. Проект демонстрирует полный пайплайн обработки естественного голоса: от распознавания аудио до генерации контекстного ответа и его синтеза в человеческую речь.

## 🛠 Стек технологий
* **Speech-to-Text (STT):** OpenAI Whisper v3 (через API Groq)
* **LLM Engine:** Llama 3 (через сверхбыстрый API Groq)
* **Text-to-Speech (TTS):** Google TTS (`gTTS`)
* **Frontend:** Streamlit (`st.audio_input` и `st.audio` с поддержкой `autoplay`)

## ⚙️ Архитектура системы (Voice Pipeline)
1. **Аудио-ввод:** Streamlit захватывает голос пользователя через микрофон браузера и передает байтовый поток (wav/mp3) на бэкенд.
2. **Транскрибация:** Модель Whisper переводит аудиопоток в идеальный текст, фильтруя шумы и акценты.
3. **Контекстная генерация (Memory):** Текст отправляется в Llama-3 вместе с историей диалога (через `st.session_state`). Бот помнит имя клиента и предыдущие вопросы, обеспечивая бесшовный диалог.
4. **Синтез речи:** Текстовый ответ LLM на лету конвертируется в аудиофайл библиотекой `gTTS` и автоматически проигрывается в браузере клиента.
